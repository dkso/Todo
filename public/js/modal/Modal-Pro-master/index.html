Paso Uno - Instalación del Plugin

Lo primero que hacemos es incluir la librería de jQuery y hacemos la llamada a nuestro plugin de la forma habitual según dicta Javascript y Jquery entre las etiquetas head. Es probable que tengamos que añadir jQuery.noConflict(); cuando sea requerido si utilizamos otras librerías como mootools que también utilizan $, en cuanto a la librería funciona con todas las versiones si encontráis cualquier bug no dudéis en decírmelo y lo arreglaré lo antes posible.

Si nos fijamos en este ejemplo básico cargamos el CSS de ejemplo para que inicialmente la ventana modal quede igual que en el ejemplo, aunque por defecto el plugin carga una configuración básica para que podáis empezar a trabajar desde el principio y una opción para que creeis vuestros propios estilos, pero no os preocupéis, eso lo miraremos más adelante, en el Paso 2.

  1. <head>
  2. <link href = " styles/modalStyle1.css " type = " text/css " rel = " stylesheet " />
  3. <script src = " http://codeorigin.jquery.com/jquery-1.10.0.min.js " type = " text/javascript " > </script>
  4. <script type = " text/javascript " >
  5. jQuery ( document ) .ready ( function ) {
  6. jQuery ( ".my_modal" ) .modalPro ();
  7. });
  8. </script>
  9. </head>

El segundo paso es escribir el HTML de la forma habitual, ya sabes entre las etiquetas body como mostramos a continuación. Todas las clases o id del HTML los podemos customizar, más adelante hablaremos de ello, y cada ventana modal tendrá que tener un selector diferente para que no haya conflictos entre las ventanas modales, a excepción del selector de estilo general que os enseñaré a utilizar en el Paso 2.

  1. <body>
  2. <div class = " my_modal " >
  3. <--! Aquí nuestros comentarios ... -->
  4. <button class = " agree_modal " > Aceptar </button>
  5. <button class = " customButton " > Custom Button </button>
  6. <button class = " customButton2 " > Custom Button2 </button>
  7. </div>
  8. <--! Ponemos nuestro botón para ejecutar la ventana modal ... -->
  9. <button class = " modaLunch " > Lanzar Ejemplo </button>
  10. </body>

Paso Dos - Opciones Generales

En este paso vamos a ver todas las opciones de las que disponemos en este plugin, desde configurar nuestros propios estilos, cambiar la posición de la ventana modal, añadir un título, añadir nuestras propias funciones a los botones personalizados, cambiar textos, profundidades, etc.

  1. <head>
  2. <link href = " youtCustomStyle.css " type = " text/css " rel = " stylesheet " />
  3. <script src = " http://codeorigin.jquery.com/jquery-1.10.0.min.js " type = " text/javascript " > </script>
  4. <script type = " text/javascript " >
  5. jQuery ( document ) .ready ( function ) {
  6. jQuery ( ".my_modal" ) .modalPro ({
  7. modaLunch: '.botonLanzaVentana', // Clase o identificador que lanza la ventana modal.
  8. overlayActive: true, // true o false sirve para activar o desactivar el fondo que está debajo de la ventana.
  9. overlayZindex: 555, // Profundidad del fondo.
  10. overlayColor: 'rgba(0, 0, 0, 0.8)', // Color de fondo, acepta rgba(), rgb(), hexagesimal, hsla().
  11. width: 400, // Ancho de la ventana.
  12. height: 400, // Alto de la ventana.
  13. top: 40, // Posición respecto al borde superior de la ventana.
  14. left: 50, // Posición respecto al borde izquierdo de la ventana.
  15. modalZindex: 999, // Profundidad del fondo de la ventana.
  16. modalBackground: '#FFF', // Color de fondo de la ventana, acepta rgba(), rgb(), hexagesimal, hsla().
  17. modalOverflow: 'auto', // Determinamos si queremos scroll dentro de la ventana modal.
  18. modalStyle: 'tuClasePersonalizada', // Con este selector aplicas tu estilo personalizado a la ventana.
  19. modalTitle: 'Escribe un título', // Escribe un título personalizado para la ventana.
  20. openInit: true, // true, false, Activamos o desactivamos la ventana al entrar en la página.
  21. closeModalButton: '.cerrarVentana', // Selector para el botón cerrar.
  22. textCloseButton: 'X', // Pon lo que quieras para cerrar la ventana, por ejemplo la típica X.
  23. seundaryClassButton: '.claseBotonPersonalizado1', // Selector para el botón personalizado, personalízalo como quieras.
  24. secundaryCallbackButton: function() { // tu código }, // Escribe tus propias funciones personalizadas para este botón.
  25. seundaryClassButton2: '.claseBotonPersonalizado2', // Selector para el botón personalizado, personalízalo como quieras.
  26. secundaryCallbackButton2: function() { // tu código } // Escribe tus propias funciones personalizadas para este botón.
  27. });
  28. });
  29. </script>
  30. </head>

Puedes cambiar estos parámetros a tu antojo para personalizar cada una de tus ventanas modales independientemente unas de otras, y puedes poner todas estás opciones o dejarlas por defecto, la única obligatoria es "modaLunch", ya que cada ventana tiene que tener su propio botón disparador para que no haya conflicto, el resto de opciones son todas opciones, ponlas todas, escoge solo algunas o deja las que están por defecto, tu tienes eliges.

Es importante que te fijes como están escritas las opciones, algunas llevan comas y otras no, si no respetamos como están escritas, seguramente el plugin no funcionará.
